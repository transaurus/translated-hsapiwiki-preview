## プレイヤーバージョン

### 2.X

バージョン`2.0`以上のプレイヤーが現在本番環境で提供されています。

[プロジェクトタイプ](/wiki/spec/overview#viewing-types)は全てのウェブプレイヤーバージョン間で共有されます。

:::info

後から追加されたパラメータ（プロジェクトIDやトレイルキャップなど）は、
まだウェブプレイヤーには実装されていません。

:::

## アプリバージョン

以下のアプリバージョンはプロジェクト内の`version`キーに対応しています：

### &lt;&equals;24

古いバージョンでは旧式の非ウェブプレイヤーが使用されます。

### 26

### 28

カスタム`when`ルールを追加。
